Linux on the Zyxel NSA320s NAS

Hi, my name is Peter. I like to tinker, its just the way i do things. I buy something, use it for a while, and take it apart to see how it works. So naturally, when i bought a NAS box already used and in bad condition, my first thought was to modify it somehow.

First i applied some homebrew patches that didnt really go that deep into the system. But then i fould a blog of someone who managed to get debian running on the damned thing. But debian is bloated, i thought to myself. So i set out on a journey to make a custom linux based firmware / operating system for my NAS.

Well. Actually this is the second NAS. I managed to brick the first one, possibly by shorting the UART pins somehow. It still boots but i get no output over serial so either the circuit board or the cpu is damaged. So i bought another one with almost the same specs. (Its still an old piece of junk)
